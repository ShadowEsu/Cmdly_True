# Cmdly — Marketing Video Scripts

Three formats: **30s Reel**, **60s Demo**, **15s Hook**.
Each scene specifies the exact app snapshot to capture and how to stage it.

---

## Snapshot Capture Guide

Before filming, capture these screens in **dark mode** on a real device or simulator.
Use iPhone 15 Pro frame (393×852 pt) — tall enough to show cards without crowding.

| ID | Screen | Setup notes |
|----|--------|-------------|
| `S1` | Home — constellation | Let the constellation animate for 3s, then pause. Commands floating, "Vault" centred. |
| `S2` | Home — stat chips + platform rail | Scroll just past the constellation so the three stat chips and coloured platform pills are visible. |
| `S3` | Home — Featured picks scroll | Scroll to "Featured picks" row. Show 2.5 cards (third card cropped = scroll invitation). |
| `S4` | Search — empty state | Search bar focused, keyboard dismissed, filter pills across the top. |
| `S5` | Search — live query "git" | Type "git" — results list populating. Cursor/ChatGPT cards visible. |
| `S6` | Platform / Claude | Open Claude deck. Burnt-orange gradient header (`from-[#4a2618]`). Several command rows visible. |
| `S7` | Platform / Gemini | Open Gemini deck. Deep-blue gradient header. |
| `S8` | Platform / Lovable | Open Lovable deck. Hot-pink gradient header. |
| `S9` | CommandRow — expanded | On any platform page, expand one command row to show the detail + "Copy details" + "Save" buttons. |
| `S10` | Collections — saved commands | Add 3–4 commands then open Collections. Dark surface, violet saved badges visible. |
| `S11` | Home — dark hero full-bleed | Full home screen, particle grid subtly visible, "Hello, Coder!" headline, constellation below. |

Shoot each at **3× resolution**. Export as PNG.
Record short **2–4 second screen recordings** of S1 (constellation drift), S5 (typing animation), S9 (expand accordion) — these are the motion moments.

## Version 1 — 30-Second Reel
*Target: TikTok · Instagram Reels · YouTube Shorts*

```
[0:22 – 0:26]  SNAPSHOT S10

## Version 2 — 60-Second Demo

  "Hello, Coder!" springs into view.
  
  VOICEOVER (or caption):
  "You switch between Claude, ChatGPT, Cursor, and Gemini every day."
```

```
[0:04 – 0:09]  SCREEN RECORDING S1

  Tap one — smooth spring navigation to platform page.
  
  VOICEOVER:
  "But you can never remember the exact commands."
```

```
[0:09 – 0:14]  SNAPSHOT S2
  Stat chips animate in: "187 Commands · 10 Platforms · 7 Signals"

  VOICEOVER:
  "Cmdly is the command vault for every AI tool you use."

```

```
[0:45 – 0:52]  SNAPSHOT S3

  Cards slide into frame.
  
  VOICEOVER:
  "New commands surface every time you open the app —
   weighted picks, trending shuffles, recommended for you."

```
[0:52 – 1:00]  SNAPSHOT S11 → logo lock-up
  Back to full dark home.
  Logo "CMDLY." pulses in, then full-screen title card.
  
  VOICEOVER:

**Voiceover style:** calm, confident, 1.0× pace. No hype, just facts.

## Version 3 — 15-Second Hook
*Target: Instagram Story · Paid social ad · App Store preview loop*
*Aspect: 9:16 vertical · Goal: single strong impression*

```
[0:00 – 0:02]  Black screen
  White text fades in:
  "How many AI commands do you actually know?"
```

```
[0:02 – 0:08]  SCREEN RECORDING S1
  Constellation explodes into view — dozens of command labels drift
  in orange, green, blue, pink, teal across the dark canvas.
  
  No caption. Let the visual speak.
```

```
[0:08 – 0:12]  SNAPSHOT S6 → S7 → S8
  Three platform decks flash in rapid 1.3 s cuts.
  Each header colour change is a jolt: orange → blue → pink.
  
  CAPTION (appears on cut 2):
  "10 platforms. Every command."
```

```
[0:12 – 0:15]  Logo lock-up
  "CMDLY." in large mono on black.
  Sub-line: "Free download."
  App Store badge.
```

**Music mood:** single punchy synth stab on cut to S1, then silence.
**No voiceover** — captions only, large and bold.

## Production Checklist

- [ ] Capture all snapshots in dark mode, 3× scale
- [ ] Record screen videos with system UI hidden (Settings → Developer → Show touches on)
- [ ] Export at 1080×1920 (9:16)
- [ ] Use CapCut / Final Cut / DaVinci for edit
- [ ] Add subtle vignette overlay (black, 15% opacity, edges only) to deepen the dark aesthetic
- [ ] Colour grade: +10 contrast, −5 highlights, +8 shadows — accentuates the glassmorphism
- [ ] Add a faint particle/noise texture overlay at 5% opacity to match the in-app particle grid feel
- [ ] Captions: SF Pro Display Bold, white, slight black stroke (1 px), centred bottom-third
- [ ] End card: static frame — logo + App Store badge on solid `#0a0a0f` background
